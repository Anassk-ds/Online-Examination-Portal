# 🎓 Online Examination Portal

A modern **Online Examination Portal** built using the **MERN Stack (Frontend currently using React + Vite)** that provides a complete online examination experience for both **Students** and **Administrators**. The application offers secure authentication, exam management, automatic evaluation, coding assessments, study notes, and a responsive premium user interface.

---

## 🌟 Overview

The Online Examination Portal is designed to simplify the examination process by allowing administrators to create and manage exams while enabling students to register, take exams, view results, and maintain study notes.

The project follows a modern SaaS-inspired design using **Inter typography**, **glassmorphism**, **light/dark mode**, and a clean user experience inspired by **Linear**, **Vercel**, and **Notion**.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/Anassk-ds/Online-Examination-Portal.git
```

---

## Navigate to the Project

```bash
cd Online-Examination-Portal
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

---

## Build for Production

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```
